# 舒尔特表专注力训练应用

## 项目简介

基于航天员训练标准的舒尔特表专注力训练应用，采用太空科技主题，提供沉浸式训练体验。

## 核心功能

### 1. 专注力训练
- 5×5 数字舒尔特表（1-25）
- 两种计时模式：30秒（标准） / 25秒（挑战）
- 严格训练模式：无视觉反馈，真实模拟飞行员训练
- 实时计时器与进度显示

### 2. 历史记录
- 完整的训练历史记录
- 按时间倒序显示
- 支持清空记录
- 显示训练结果（成功/失败/超时）

### 3. 数据统计
- 训练总次数统计
- 成功率分析
- 平均用时计算
- 最佳成绩追踪
- 可视化图表展示

### 4. 评级系统
- S级（≤8秒）：航天员级别
- A级（9-15秒）：优秀
- B级（16-20秒）：良好
- C级（21-25秒）：达标
- D级（>25秒）：需练习

## 技术栈

- **前端框架**：Vue 3 + Vite
- **样式框架**：Tailwind CSS
- **路由**：Vue Router 4
- **工具库**：VueUse
- **图表库**：Chart.js + vue-chartjs
- **数据存储**：LocalStorage

## 使用说明

### 训练模式
1. 选择计时模式（30秒标准 / 25秒挑战）
2. 点击"开始训练"按钮
3. 按 1-25 顺序点击数字
4. 注意：点击错误即失败，数字不提供视觉反馈
5. 完成后查看成绩与评级

### 查看记录
1. 点击底部导航"记录"标签
2. 查看所有历史训练记录
3. 可以清空历史数据

### 数据统计
1. 点击底部导航"统计"标签
2. 查看训练数据汇总
3. 查看成绩趋势图表

## 注意事项

1. 本地存储数据保存在浏览器中，清除浏览器数据会丢失训练记录
2. 建议每日训练 10-15 分钟，避免长时间集中注意力导致视觉疲劳
3. 从 30 秒模式开始练习，达标后可挑战 25 秒模式
4. 训练时保持适当距离（屏幕距离 50-70 厘米）
