# 舒尔特表专注力训练应用

较为标准的舒尔特表专注力训练应用，采用太空科技主题，提供沉浸式训练体验。

在线体验：https://younglina.wang/shulte/training

## 核心功能

### 1. 专注力训练
- 5×5 1cm×1cm 数字舒尔特表（1-25）
- **限时模式**：25秒（挑战）/ 30秒（标准）
- **不限时模式**：无时间限制，自由练习
- 严格训练模式：无视觉反馈，真实模拟飞行员训练
- 实时计时器与进度显示
- **重新训练**：可选择历史记录重新练习

### 2. 历史记录
- 完整的训练历史记录
- 按时间倒序显示
- 支持重新训练历史记录
- 显示训练结果（成功/失败/超时）
- 重新训练时自动对比成绩进步/退步

### 3. 数据统计
- 训练总次数统计
- 成功率分析
- 平均用时计算
- **最佳成绩追踪**（总成绩 + 限时/不限时分开统计）
- 可视化图表展示
  - 训练结果分布（成功/失败/超时）
  - 成绩趋势折线图（最近20次训练）

### 4. 评级系统
- S级（≤8秒）：航天员级别
- A级（9-15秒）：优秀
- B级（16-20秒）：良好
- C级（21-25秒）：达标
- D级（>25秒）：需练习

## 使用说明

### 训练模式
1. 选择计时模式（30秒标准 / 25秒挑战 / 不限时）
2. 点击"开始训练"按钮
3. 按 1-25 顺序点击数字
4. 注意：点击错误即失败，数字不提供视觉反馈
5. 完成后查看成绩与评级

### 查看记录
1. 点击底部导航"记录"标签
2. 查看所有历史训练记录
3. 可选择记录重新训练
4. 可以清空历史数据

### 数据统计
1. 点击底部导航"统计"标签
2. 查看训练数据汇总
3. 查看成绩趋势图表

## 数据存储

数据使用 **IndexedDB** 存储

存储字段：
- `id`: 自增主键
- `timestamp`: 时间戳
- `mode`: 训练模式
- `result`: 结果（success/fail/timeout）
- `timeUsed`: 用时
- `gridNumbers`: 网格数字排列
- `replayOf`: 原始记录ID（重练时）

## 注意事项

1. 数据保存在浏览器 IndexedDB 中，清除浏览器数据会丢失训练记录
2. 建议每日训练 10-15 分钟，避免长时间集中注意力导致视觉疲劳
3. 从 30 秒模式开始练习，达标后可挑战 25 秒模式
4. 训练时保持适当距离（屏幕距离 50-70 厘米）
